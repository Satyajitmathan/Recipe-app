# RecipeHub 🍽️

A modern recipe management application built with React.

## Features

* Create Recipes
* Update Recipes
* Delete Recipes
* Add to Favorites
* Category Filtering
* Responsive UI
* Local Storage Persistence

## Tech Stack

* React
* React Router
* Context API
* Tailwind CSS
* React Hook Form
* React Toastify

## Live Demo

https://recipehub-satyajit.vercel.app/

## Installation

```bash
npm install
npm run dev
```
